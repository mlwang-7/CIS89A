This site uses a flexbox for the layout. When the screen width is 600px or greater, the left bar takes up 20% of the screen and the right side takes up 80%. Else, this side takes up 100% and goes on top, and the left side goes to the bottom.

  • The navigation bar is fixed to the top of the page.
  • Title of the page is floated left, navigation bar is floated right.
  • The pictures of bunnies to the left are also in a flexbox.
